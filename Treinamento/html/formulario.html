Características

Propriedades

  • accept-charset
  • autocomplete
  • action

  • target
accept-charset
Define o charset utilizado pelo formulário. Ex: UFT-8, UTF-16, ANSI. 

<form accept-charset="ISO-8859-1">
  <!--Campos aqui...-->
</form>
autocomplete
A medida que os dados são adicionados no formulário, o navegador sugere os valores inseridos anteriormente. 

<form autocomplete="on">
  <input id="email" type="text" />
</form>
action
Especifica o destino dos dados do formulário. 

<form action="action="http://www.nosso-servidor.com.br/cadastro/usuario.php"">
  <input id="email" type="text" />
</form>
method
Especifíca o método HTTP que será usado para transmitir os dados.
Valores possíveis são: GET, POST 

<form action="action="http://www.nosso-servidor.com.br/cadastro/usuario.php"" method="POST">
  <input id="email" type="text" />
</form>
enctype
Especifíca quando os dados do formulário precisam ser codificados antes do envio.
Valores possíveis são: application/x-www-form-urlencoded, multipart/form-data, text/plain.
Utilizado apenas quando o método de envio for POST. 

<form action="action="http://www.nosso-servidor.com.br/cadastro/usuario.php"" enctype="multipart/form-data">
  <input id="email" type="text" />
</form>
novalidate
Impede a validação do formulário antes da submissão. 

<form action="action="http://www.nosso-servidor.com.br/cadastro/usuario.php"" novalidate>
  <input id="email" type="text" />
</form>
target
Indica onde a página de destino será aberta.
Usa os mesmos valores da tag a ( anchor).
Valoes possíveis: _blank, _self, _parent, _top, framename 

<form target="_blank" >
  <input id="email" type="text" />
</form>

input

type="text"
Campo básico para colocar texto. 

<input type="text" />
  • placeholder - texto a ser exibido quando o campo está vazio.
  • maxlength - quantidade máxima de caracteres.
  • required - indica que o campo é obrigatório.
  • disabled - deixa o campo desabilitado para edição.
  • readonly - impede edições do campo.
type="email"
Campo destinado para inserção de email com validação.
type="search"
Campo destinado para pesquisa.
type="url"
Campo destinado para inserção de url com validação.
type="number"
Campo destinado para inserção de número com validação.
  • min - validação do valor mínimo a ser inserido no campo.
  • max - validação do valor máximo a ser inserido no campo.
  • step - intervalo válido entre min e max.
type="radio"
Campo que permite escolha de uma opção.
  • checked - indica que uma opção está selecionada por padrão.

<form>
  <input type="radio" name="opcao" checked /> Masculino
  <input type="radio" name="opcao" /> Feminino
</form>
Masculino Feminino
type="checkbox"
Campo que permite escolha de mais de uma opção.
  • checked - indica que uma opção está selecionada por padrão.

<form>
  <input type="checkbox" name="transparente" checked /> Transparente
  <input type="checkbox" name="azul" /> Azul
  <input type="checkbox" name="vermelho" /> Vermelho
</form>
Transparente Azul Vermelho
type="time"
Campo que permite escolha de hora.
  • min - Especifica data mínima no formato HH:MM.
  • max - Especifica data máxima no formato HH:MM.
O formato da data a ser exibido assim como a linguágem usada no datepicker depende da linguágem na qual o browser está configurado. 

<form>
  <input type="time" name="time" min="08:00"/>
</form>
type="date"
Campo que permite escolha de data.
  • min - Especifica data mínima no formato YYYY-MM-DD.
  • max - Especifica data máxima no formato YYYY-MM-DD.
O formato da data a ser exibido assim como a linguágem usada no datepicker depende da linguágem na qual o browser está configurado. 

<form>
  <input type="date" name="data" min="2018-06-08"/>
</form>
type="datetime-local"
Campo que permite escolha de data e hora local.
  • min - Especifica data mínima no formato YYYY-MM-DD.
  • max - Especifica data máxima no formato YYYY-MM-DD.

O formato da data a ser exibido assim como a linguágem usada no datepicker depende da linguágem na qual o browser está configurado.

Atualmente não suportado no Firefox 60.0.2. 


<form>
  <input type="datetime-local" name="data"/>
</form>
Terminar itens de date time
type="range"
Permite selecionar um valor dentro de limites definidos.
  • min - Especifica o valor mínimo.
  • max - Especifica o valor máximo.

<form>
  <input type="range" min="0" max="10"/>
</form>
type="color"
Permite selecionar cor através da paleta de cores. 

<form>
  <input type="color"/>
</form>
type="file"
Permite selecionar arquivos. 

<form>
  <input type="file"/>
</form>
type="button"
Criando botões.
  • submit - botão usado para enviar um formulário.
  • reset - botão usado para limpar os campos de um formulário.

<form>
  <input type="button"/>
</form>
textarea
Campo para inserir texto em múltiplas linhas. 

<textarea rows="15" cols="20">Texto inicial</textarea>
  • rows - número de linhas do componente.
  • cols - número de colunas do componente.
select
Permite selecionar um item dentro de uma lista de opções.
  • multiple - permite a seleção/exibição de mais de um item.

<select>
  <option value="boleto">Boleto</option>
  <option value="cartao">Cartão de Crédito</option>
  <option value="crediario">Crediário</option>
</select>
datalist
  • list - id do componente datalist.

Permite selecionar um item dentro de uma lista de opções.


<input list="items" name="shopping-list">
  <datalist id="items">
  <option value="Aveia">
  <option value="Manteiga">
  <option value="AçucarMascavo">
</datalist>
fieldset

Permite criar divisões no formulário para organização de partes lógicas no cadastro.


<fieldset>
    <legend>Dados Usuário</legend>
    Nome: <input type="text"/><br />
    Email: <input type="text"/><br />
</fieldset>
Dados Usuário Nome:
Email:
label

Permite dar nome a campos de um formulário. Habilita a funcionalidade de autofocus ao clicar no label o input exibe o cursor. 


<fieldset>
    <legend>Dados Usuário</legend>
    Nome: <input type="text"/><br />
    Email: <input type="text"/><br />
</fieldset>
output

Permite exibir o resultado de alguma operação.


<fieldset>
    <legend>Dados Usuário</legend>
    Nome: <input type="text"/><br/>
    Email: <input type="text"/><br/>
</fieldset>
