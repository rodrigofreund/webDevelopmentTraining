Propriedades de um formulário

Propriedades

accept-charset

Define o charset utilizado pelo formulário. Ex: UFT-8, UTF-16, ANSI. 

<form accept-charset="ISO-8859-1">
  <!--Campos aqui...-->
</form>

autocomplete

A medida que os dados são adicionados no formulário, o navegador sugere os valores inseridos anteriormente. 

<form autocomplete="on">
  <input id="email" type="text" />
</form>

action

Especifica o destino dos dados do formulário. 

<form action="action="http://www.nosso-servidor.com.br/cadastro/usuario.php"">
  <input id="email" type="text" />
</form>

method

Especifíca o método HTTP que será usado para transmitir os dados.
 Valores possíveis são: GET, POST 

<form action="action="http://www.nosso-servidor.com.br/cadastro/usuario.php"" method="POST">
  <input id="email" type="text" />
</form>

enctype

Especifíca quando os dados do formulário precisam ser codificados antes do envio.
 Valores possíveis são: application/x-www-form-urlencoded, multipart/form-data, text/plain.
 Utilizado apenas quando o método de envio for POST. 

<form action="action="http://www.nosso-servidor.com.br/cadastro/usuario.php"" enctype="multipart/form-data">
  <input id="email" type="text" />
</form>

novalidate

Impede a validação do formulário antes da submissão.
 

<form action="action="http://www.nosso-servidor.com.br/cadastro/usuario.php"" novalidate>
  <input id="email" type="text" />
</form>

target

Indica onde a página de destino será aberta.
 Usa os mesmos valores da tag a (anchor).
 Valoes possíveis: _blank, _self, _parent, _top, framename 

<form target="_blank" >
  <input id="email" type="text" />
</form>

Elementos

input

type="text"

Campo básico para colocar texto. 

<input type="text" />
  • placeholder - texto a ser exibido quando o campo está vazio.
  • maxlength - quantidade máxima de caracteres.
  • required - indica que o campo é obrigatório.
  • disabled - deixa o campo desabilitado para edição.
  • readonly - impede edições do campo.

type="email"

Campo destinado para inserção de email com validação.

type="search"

Campo destinado para pesquisa.

type="url"

Campo destinado para inserção de url com validação.

type="number"

Campo destinado para inserção de número com validação.
  • min - validação do valor mínimo a ser inserido no campo.
  • max - validação do valor máximo a ser inserido no campo.
  • step - intervalo válido entre min e max.

type="radio"

Campo que permite escolha de uma opção.
  • checked - indica que uma opção está selecionada por padrão.

<form>
  <input type="radio" name="opcao" checked /> Masculino
  <input type="radio" name="opcao" /> Feminino
</form>
Masculino Feminino

textarea

Campo para inserir texto em múltiplas linhas. 

  <textarea rows="15" cols="20">Texto inicial</textarea>
  • rows - número de linhas do componente.
  • cols - número de colunas do componente.
