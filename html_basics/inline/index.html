Exemplo usando blocos inline

Imagem alinhada

Como imagens são, por padrão, blocos inline. Vamos ver o que acontece quando você adiciona uma imagem diretamente em uma linha de texto.

Este paragrafo contém uma imagem muito linda que fica junto ao texto. esta imágem usa as dimenções intrísecas por padrão. As caixas de linha são altas o suficiente para conter as caixas de cada elemento inline da linha.

Imagens in line pode ser utilizado vertical-align para alinhar a imagem como esta na linha usando middle.

Esta imagem encontra-se alinhada no topo.
