Usando Float

A propriedade float foi especificada para que o texto envolvesse uma imagem. Usando esta propriedade juntamente com o padrão width:auto transforma-a numa caixa retrátil. Usando a propriedade float faz com que o elemento toque o limite da caixa de conteúdo ou outra flutuação. Além disto, flutuar o elemento faz com que ele seja removido do fluxo do do documento, assim ele não ocupa nenhum espaço. Isto significa que os elementos block-level subsequentes que não estão flutuando ignoram o elemento flutuante e sobem para preencher a lacuna.
